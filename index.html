kotlinx.coroutines

kotlinx.coroutines reference documentation

Library support for Kotlin coroutines.

Modules

  • kotlinx-coroutines-core core primitives to work with coroutines.
  • kotlinx-coroutines-jdk8 with additional libraries for JDK8 (or Android API level 24).
  • kotlinx-coroutines-nio with extensions for asynchronous IO on JDK7+.
  • kotlinx-coroutines-swing with Swing context for Swing UI applications.
  • kotlinx-coroutines-javafx with JavaFx context for JavaFX UI applications.
  • kotlinx-coroutines-rx with utilities to build Observable objects from RxJava. It is in very basic form now (example-only, not even close to production use)
